## 82.1 路径记忆的本质

**定义 82.1** (路径记忆张量):

$$
M_{\mu\nu} = \int_{\text{path}} \psi^*(\tau) \frac{\partial}{\partial x^\mu}\frac{\partial}{\partial x^\nu} \psi(\tau) d\tau
$$

**定理 82.1**: 路径完整编码其历史。

*证明*:
通过路径积分重构：
$$
\text{Path} = \arg\max_{\gamma} \int M_{\mu\nu}dx^\mu dx^\nu
$$
记忆张量唯一确定原路径。∎
